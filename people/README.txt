HOW TO ADD A NEW PERSON
=======================

1. Create a folder inside "people/" named after the person.
   Example: people/john/

2. Add exactly TWO photos inside that folder:
   - photo.jpg      → their licence face photo
   - signature.jpg  → their signature photo

3. Share this link with them:
   https://vicroad-app.github.io/VicRoad/?id=john

That's it! Replace "john" with whatever folder name you used.

EXAMPLE STRUCTURE:
people/
  john/
    photo.jpg
    signature.jpg
  sarah/
    photo.jpg
    signature.jpg

LINKS:
  ?id=john  → shows john's photos
  ?id=sarah → shows sarah's photos
