# VicRoad

Static export of the Blitz ID web app.

## Run locally

1. Start a local static server from the repository root:

```bash
python3 -m http.server 8000
```

2. Open the app entry page:

```text
http://127.0.0.1:8000/index.html
```

`index.html` is the stable entry point for local hosting and static platforms.
