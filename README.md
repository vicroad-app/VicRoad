# VicRoad

A project for VicRoad-related functionality.

## Getting Started

### Adding Your Code

If you have code in a zip folder, you can add it to this repository by:

1. **Extracting the zip file** on your computer
2. **Copying the extracted files** into your local clone of this repository
3. **Committing and pushing** the files:

```bash
git add .
git commit -m "Add project code"
git push
```

Alternatively, you can upload files directly via the GitHub web interface by:

1. Navigating to the repository on GitHub
2. Clicking **Add file** → **Upload files**
3. Dragging and dropping your files (or extracting and uploading your zip contents)
